# Equity Valuation Approach

A valuation method that focuses on the cash flows available to equity holders (FCFE) after all debt payments. It explicitly considers new debt issues and debt repayments. This approach should yield the same equity value as the firm valuation approach if consistent assumptions about financial leverage are made.
