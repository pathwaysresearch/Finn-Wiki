# Creative Destruction

The process by which new innovations systematically replace older technologies and the firms built around them, generating new products while rendering existing capabilities obsolete. A process that incumbents will rationally resist, and which requires certain institutional prerequisites to operate at the societal level. A process where new technologies or innovations replace existing ones, which incumbents may rationally try to block or delay.
