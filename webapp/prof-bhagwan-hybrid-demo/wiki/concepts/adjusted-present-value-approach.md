# Adjusted Present Value Approach

A valuation method that values a firm by first estimating its value with no leverage, then adding the present value of interest tax savings, and subtracting expected bankruptcy costs. It presumes it's easier to compute the valuation impact of debt in absolute terms rather than proportional terms. The original approach discounted tax benefits using the cost of debt.
