# H Model

A two-stage growth model that is a variant of the classical two-stage model, designed for computational simplicity. Unlike the classical model, the growth rate in the initial phase is not constant but declines linearly over time to reach a stable growth rate. This model avoids a precipitous drop in growth rates but assumes a strict linear decline and a constant payout ratio, which can be inconsistent.
