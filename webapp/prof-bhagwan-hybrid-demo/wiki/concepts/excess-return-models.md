# Excess Return Models

A variant of the Discounted Cash Flow (DCF) approach that separates cash flows into 'excess return cash flows' and 'normal return cash flows.' The value of a firm is computed as a function of these expected excess returns. Economic Value Added (EVA) is presented as a widely used variant of this model. A class of valuation models built on the principle that value is created by generating returns on investments that exceed their cost. These models link reinvestment to future growth, implying that higher future earnings require reinvestment in new assets or more efficient use of existing ones. They can converge with discounted cash flow models if assumptions about growth and reinvestment are consistent.
