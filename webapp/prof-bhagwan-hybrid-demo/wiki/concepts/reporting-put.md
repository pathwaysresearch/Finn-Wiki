# Reporting Put

A mechanism of manipulation where an agent smooths a reported number without changing the underlying reality, which compresses only the left tail of the distribution. A misreporting mechanism where an agent selectively smooths negative outcomes to compress the left tail of the reported distribution, while the true underlying outcome remains unchanged.
