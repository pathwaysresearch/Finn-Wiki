## Abstract

This is a transcript of Session 16 from MIT 15.401 (Finance Theory I), Fall 2008, taught by Professor Andrew Lo. The lecture continues the discussion of the risk-reward trade-off, deriving and explaining the Security Market Line (SML) and the Capital Asset Pricing Model (CAPM), which provide the framework for calculating the appropriate discount rate for any project or security. The lecture distinguishes between the Capital Market Line and the Security Market Line, explains the role of beta in determining expected returns, and discusses the significance of the tangency portfolio in portfolio theory.

## Key Claims

- The Security Market Line applies to every single security in the investment universe, unlike the Capital Market Line which applies only to efficient portfolios
- When an asset's beta equals 1, its expected rate of return equals the market's expected rate of return; when beta equals 0, the expected return equals the risk-free rate
- A beta of 0 does not mean the asset has no volatility — it means the asset's risk is fully diversifiable
- You would never want to hold just IBM alone because mixing it with other assets always allows higher expected return for the same risk, or lower risk for the same expected return
- An asset negatively correlated with the tangency portfolio is tremendously valuable, but assets with negative beta and positive expected return are very rare in practice
