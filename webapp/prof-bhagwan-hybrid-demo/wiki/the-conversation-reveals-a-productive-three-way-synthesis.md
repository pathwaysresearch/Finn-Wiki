# Language Barriers as Financial Distance

A productive three-way synthesis: gravity model residuals, arbitrage deviations, and home bias all converge as measures of the same underlying information friction.

Language barriers represent a non-geographic distance variable that operates mathematically identically to physical distance in the gravity framework — but with an asymmetric incidence that disproportionately harms developing-market entrepreneurs seeking foreign capital.

The fintech angle — AI-driven multilingual financial analysis compressing language-based information distance — is a novel connection between the FinTech for Billions agenda and the home bias literature that is not currently captured in the Wiki.
